This project uses yt-dlp in the cloud to download youtube video's. It's a bit slow since it uses the free tier of railway, but it works. To install: go to "code" and click: "download zip". Unzip it and go to chrome://extensions. Enable developer mode and click "load unpacked". Select the folder you just unzipped. After it's installed, you don't need to do anything yourself, just press the download button. The server git repository is: https://github.com/thecoolgamer551/extension-only-downloader-server-side
